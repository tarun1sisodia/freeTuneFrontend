## Reporting a Vulnerability

We take the security of `freeTune` seriously. If you have found a vulnerability, please report it to us immediately.

### **How to Report**
Please **DO NOT** create a public GitHub issue. Instead, report the vulnerability via:

1.  **Email**: Send the details to [coccoder999@gmail.com](mailto:coccoder999@gmail.com).
2.  **Private Disclosure**: If enabled, use the "Report a vulnerability" button in the **Security** tab of this repository.

### **What to Include**
*   Description of the vulnerability.
*   Steps to reproduce.
*   Potential impact.

### **Our Process**
1.  We will acknowledge your report within **48 hours**.
2.  We will investigate and provide an estimated timeline for a fix.
3.  Once fixed, we will release a patch and credit you (with permission).
